======================================
MaskRCNN for NuCLS Dataset
======================================

This project implements the MaskRCNN architecture for nuclei instance segmentation using the NuCLS dataset. It includes tools for data preparation, model training, and inference on histopathology images of breast cancer from The Cancer Genome Atlas (TCGA).
